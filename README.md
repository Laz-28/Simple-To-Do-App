# Simple ToDo App

A lightweight, browser-based task management application built with vanilla JavaScript. This project was created as a foundational exercise to practice DOM manipulation, event handling, and basic front-end development.

## 🚀 Features

*   **Add Tasks:** Easily input and append new items to your daily list.
*   **Remove Tasks:** Delete specific items once they are completed or no longer needed.
*   **Clean UI:** A simple, intuitive design focused on core functionality without unnecessary clutter.

## 🛠️ Technologies Used

*   **HTML5:** Provides the structure and semantic layout of the application.
*   **CSS3:** Handles the visual styling and layout logic.
*   **JavaScript (Vanilla):** Powers the interactive logic, including listening for user input and dynamically updating the webpage.

## 💻 How to Run Locally

Because this project uses vanilla web technologies, no build tools or package managers are required to run it.

1.  Clone this repository to your local machine (or download the ZIP file).
2.  Navigate to the project directory.
3.  Double-click the `index.html` file to open it in your default web browser.
4.  Start adding your tasks!
